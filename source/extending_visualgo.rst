Extending Visualgo
==================


.. toctree::
    adding_data_structure

Once your new data structure is beautiful done, you can link it with a visual structure.
You have to choices :

- Use an existing `VisualWidget` that fulfill your visual need. You can optionally add a new constructor.
- Create your own `VisualWidget`.
